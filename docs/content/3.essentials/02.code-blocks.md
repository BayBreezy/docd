## Basic

### Inline code

Use inline code for short references inside text.

::prose-show-case
---
prose: true
---
`content.config.ts`

#code
```mdc
`content.config.ts`
```
::

### Code blocks

Use fenced code blocks for multi-line examples with syntax highlighting.

::prose-show-case
---
prose: true
---
```ts
export default defineNuxtConfig({
  extends: ["docd"],
})
```

#code
````mdc
```ts
export default defineNuxtConfig({
extends: ["docd"],
})
```
````
::

### Filenames

Add a filename in square brackets to give the snippet more context.

::prose-show-case
---
prose: true
---
```ts [nuxt.config.ts] noFormat
export default defineNuxtConfig({
  extends: ["docd"],
})
```

#code
````md
```ts [nuxt.config.ts] noFormat
export default defineNuxtConfig({
extends: ["docd"],
})
```
````
::

Every Docd code block includes a copy button automatically.

## Advanced

### `prose-code-group`

Use `prose-code-group` to show equivalent snippets in tabs.

::prose-show-case
---
prose: true
class: "[&>div]:*:my-0 [&>div]:*:w-full"
---
  :::prose-code-group{.w-full}
  ```bash [pnpm]
  pnpm add docd
  ```

  ```bash [yarn]
  yarn add docd
  ```

  ```bash [npm]
  npm install docd
  ```

  ```bash [bun]
  bun add docd
  ```
  :::

### `prose-code-tree`

Use `prose-code-tree` when you want to show a small project structure or multiple related files.

::prose-show-case
---
prose: true
class: "[&>div]:*:my-0 [&>div]:*:w-full"
---
  :::prose-code-tree{default-value="nuxt.config.ts"}
  ```ts [nuxt.config.ts]
  export default defineNuxtConfig({
    extends: ["docd"],
    site: {
      name: "My Docs",
    },
  })
  ```

  ```ts [app/app.config.ts]
  export default defineAppConfig({
    docd: {
      ui: {
        header: {
          title: "My Docs",
        },
      },
    },
  })
  ```

  ```md [content/index.md]
  ---
  title: Home
  description: Welcome to my documentation.
  ---

  # Welcome
  ```

  ```json [package.json]
  {
    "name": "my-docs",
    "private": true,
    "scripts": {
      "dev": "nuxt dev",
      "build": "nuxt build"
    },
    "dependencies": {
      "docd": "^0.0.1",
      "nuxt": "^4.4.2"
    }
  }
  ```
  :::

### `prose-code-collapse`
